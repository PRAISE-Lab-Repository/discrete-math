Here is how you can enter the logic symbols:

  • - Use '^' for AND (Conjunction)
  • - Use 'v' for OR (Disjunction)
  • - Use '~' for NOT (Negation)
  • - Use '->' for IMPLIES
  • - Use '<->' for BICONDITIONAL
  • - Use 'T' for TRUE
  • - Use 'F' for FALSE

Note: Make sure to use lowercase for the operators and uppercase for boolean constants.
